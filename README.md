# Elige tu Poké Ball

Proyecto React para elegir entre 3 Poké Balls; cada una contiene un Pokémon inicial (estilo Pokémon Rubí).

## Cómo usar

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Arrancar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre la URL que muestre la terminal (por ejemplo `http://localhost:5173`).

4. Haz clic en una Poké Ball para seleccionarla y luego en **"¡Elegir a [Pokémon]!"** para confirmar. Verás el Pokémon elegido con su sprite.

## Contenido de cada bola

- **Poké Ball 1:** Bulbasaur (Planta/Veneno)
- **Poké Ball 2:** Charmander (Fuego)
- **Poké Ball 3:** Squirtle (Agua)

Hecho con React + Vite. Estética pixel art inspirada en los juegos clásicos de Pokémon.
